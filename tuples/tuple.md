## Tuples in Python
- 튜플은 리스트와 거의 흡사함
- 리스트는 [ ]로, 튜플은 ( )로 둘러싸는 특징을 가짐
- 리스트는 그 값의 생성/삭제/수정이 가능하지만 **_튜플은 그 값을 바꿀 수는 없음_** ✔✔✔
- 아래의 tuple2처럼, 요소를 1개만 가질때에는 콤마를 반드시 붙여아함
- 튜플은 tuple4처럼, 괄호를 생략할 수 있음

```python
tuple1 = ()
tuple2 = (1, )
tuple3 = (1, 2, 3)
tuple4 = 1, 2, 3
tuple5 = ('a', 'b', ('ab', 'cd'))
```
