## Bool이란?
- 참(True)과 거짓(False)을 나타내는 자료형 
- 파이썬에서 True와 False는 예약어로 첫문자를 항상 대문자로 사용해야함

### 자료형의 참과 거짓
-> _항상 기억해야 할 부분은 None은 거짓을 반환함을 알아두어야 한다_

| 자료형  | 값              | 참 or 거짓 |
|------|----------------|---------|
| 문자열  | "python"       | 참       |
|      | "" (값이 비어있는 것) | 거짓      |
| 리스트  | [1, 2, 3]      | 참       |
|      | []             | 거짓      |
| 튜플   | ()             | 거짓      |
| 딕셔너리 | {}             | 거짓      |
| 숫자형  | 0이 아닌 숫자       | 참       |
|      | 0              | 거짓      |
|      | None           | 거짓      |
